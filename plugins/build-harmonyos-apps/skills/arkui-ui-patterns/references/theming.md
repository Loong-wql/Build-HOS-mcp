# Theming And Responsive Layout

## Defaults
- Prefer existing project tokens for color, spacing, type, radius, and motion.
- Check text expansion for Chinese and English strings.
- Define behavior for phone, tablet, foldable, and landscape only when those targets are in scope.
- Keep touch targets clear and consistent.

## Checks
- Does the UI work in light and dark themes if the app supports both?
- Does critical text wrap or truncate intentionally?
- Are primary and secondary actions visually distinct?
