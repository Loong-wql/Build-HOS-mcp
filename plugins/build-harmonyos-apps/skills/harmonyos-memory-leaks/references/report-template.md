# Memory Report Template

## Flow
Exact reproduction steps and expected release point.

## Evidence
Device/emulator, build mode, profiler/log artifacts, before/after captures.

## Findings
For each retained object or resource:
- intended lifetime;
- suspected retaining edge;
- evidence;
- smallest fix;
- verification result.

## Remaining Risk
List unverified profiler gaps, SDK/runtime noise, or flows not recaptured.
